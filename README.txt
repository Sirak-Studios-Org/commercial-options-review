# Commercial Options Review – Vercel Package

This package is ready to deploy to Vercel as a static site.

## Contents
- `public/index.html` — combined Miami + Atlanta interactive map app
- `vercel.json` — Vercel routing so the app loads correctly at the root domain
- `index.html` — fallback redirect file

## Deploy on Vercel
1. Unzip this folder.
2. Create a new project in Vercel.
3. Upload the unzipped folder, or import it into a repo and connect that repo to Vercel.
4. Deploy.

## Result
Your root domain will load the combined Miami + Atlanta map experience.
